      2.Prenumele: "Eugeniu" --> Second stage
      3.Telefon: "Nr de telefon"
      4.Orasul: "Numele orasului"
      5.Virsta: "Virsta persoanei"
  3.Cautare
    1.Va cauta persoanele in fisierul "phone.txt"
